# Pradhan Mantri Janaushadhi Generic Medical Store — Official Website

> **Quality Generic Medicines at Up to 90% Less Price · Vadodara**

A modern, responsive single-page website for the **Pradhan Mantri Bhartiya Janaushadhi Kendra (PMBJK)** store in Vadodara — India's Government-backed initiative for affordable healthcare.

🌐 **Live Site:** [janaushadhi-store.github.io](https://janaushadhi-store.github.io) *(update after deployment)*

---

## 📋 About

This website serves as the official digital presence for the Janaushadhi Generic Medical Store in Vadodara, Gujarat. It helps local citizens discover the benefits of generic medicines and locate the store easily.

### Key Features
- Responsive design (mobile + desktop)
- Price comparison table (branded vs. generic)
- Product categories with savings indicators
- Live savings ticker banner
- FAQ accordion
- Google Maps integration
- Scroll animations & modern UI

## 🚀 Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, `/ (root)` folder
4. Click **Save** — your site will be live at `https://<username>.github.io/<repo-name>/`

## 🏛️ About PMBJP

The **Pradhan Mantri Bhartiya Janaushadhi Pariyojana (PMBJP)** is an initiative of the Government of India to make quality generic medicines available at affordable prices. Medicines are sourced from WHO-GMP certified manufacturers and tested in NABL-accredited labs.

Learn more: [janaushadhi.gov.in](https://janaushadhi.gov.in)
